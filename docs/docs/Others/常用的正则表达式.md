# 常用的正则表达式

## 数字类

```js
// 只能输入数字
/^\d+$/

// 只能输入n个数字
/^\d{n}$/

// 只能输入n个以上数字
/^\d{n,}$/

// 只能输入m-n个数字
/^\d{m,n}$/

// 零和非零开头的数字
/^(0|[1-9][0-9]*)$/

// 非零开头的最多带两位小数的数字
/^([1-9][0-9]*)+(.[0-9]{1,2})?$/

// 带1-2位小数的正数或负数
/^(\-)?\d+(\.\d{1,2})?$/

// 正数、负数、和小数
/^(\-|\+)?\d+(\.\d+)?$/

// 有两位小数的正实数
/^[0-9]+(.[0-9]{2})?$/

// 有1~3位小数的正实数
/^[0-9]+(.[0-9]{1,3})?$/

// 非零的正整数
/^[1-9]\d*$/
/^([1-9][0-9]*){1,3}$/
/^\+?[1-9][0-9]*$/

// 非零的负整数
/^\-[1-9][]0-9*$/
/^-[1-9]\d*$/

// 非负整数
/^\d+$/
/^[1-9]\d*|0$/

// 非正整数
/^-[1-9]\d*|0$/
/^((-\d+)|(0+))$/

// 非负浮点数
/^\d+(\.\d+)?$/
/^[1-9]\d*\.\d*|0\.\d*[1-9]\d*|0?\.0+|0$/

// 非正浮点数
/^((-\d+(\.\d+)?)|(0+(\.0+)?))$/
/^(-([1-9]\d*\.\d*|0\.\d*[1-9]\d*))|0?\.0+|0$/

// 正浮点数
/^[1-9]\d*\.\d*|0\.\d*[1-9]\d*$/
/^(([0-9]+\.[0-9]*[1-9][0-9]*)|([0-9]*[1-9][0-9]*\.[0-9]+)|([0-9]*[1-9][0-9]*))$/

// 负浮点数
/^-([1-9]\d*\.\d*|0\.\d*[1-9]\d*)$/
/^(-(([0-9]+\.[0-9]*[1-9][0-9]*)|([0-9]*[1-9][0-9]*\.[0-9]+)|([0-9]*[1-9][0-9]*)))$/

// 浮点数
/^(-?\d+)(\.\d+)?$/
/^-?([1-9]\d*\.\d*|0\.\d*[1-9]\d*|0?\.0+|0)$/
```

## 字符类

```js
// 匹配行首行尾空白
/(^\s*)|(\s*$)/

// 空白行
/\n\s*\r/

// 匹配中文字符
/[\u4E00-\u9FA5]|[\uFE30-\uFFA0]/gi

// 双字节字符
/[^\x00-\xff]/igm

// 只能是英文字母
/^[a-z]+$/i

// 只能是大写英文字母
/^[A-Z]+$/

// 只能是小写
/^[a-z]+$/

// 只能英文和数字
/^[A-Za-z0-9]+$/i

// 只能英文、数字、下划线
/^\w+$/

// 中文、英文、数字包括下划线
/^[\u4E00-\u9FA5A-Za-z0-9_]+$/

// 中文、英文、数字但不包括下划线等符号
/^[\u4E00-\u9FA5A-Za-z0-9]+$/
/^[\u4E00-\u9FA5A-Za-z0-9]{2,20}$/

// 可以输入含有^%&',;=?$\"等字符
/[^%&',;=?$\x22]+/

// 禁止输入含有~的字符
/[^~\x22]+/
```

## 常用表单

```js
// email
/\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*/

// url地址
/^https?:\/\/(([a-zA-Z0-9_-])+(\.)?)*(:\d+)?(\/((\.)?(\?)?=?&?[a-zA-Z0-9_-](\?)?)*)*$/i

// 手机号码
/^(0|86|17951)?(13[0-9]|15[012356789]|166|17[3678]|18[0-9]|14[57])[0-9]{8}$/

// 固定号码
/^(\(\d{3,4}-)|\d{3.4}-)?\d{7,8}$/

// 身份证号码
/^(^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$)|(^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])((\d{4})|\d{3}[Xx])$)$/

// 邮政编码
/^[1-9]\d{5}(?!\d)$/

// 字母开头，允许5-16字节，允许字母数字下划线
/^[a-zA-Z][a-zA-Z0-9_]{4,15}$/

// 以字母开头，长度在6~18之间，只能包含字母、数字和下划线
/^[a-zA-Z]\w{5,17}$/

// 必须包含大小写字母和数字的组合，不能使用特殊字符，长度在8-10之间
/^(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,10}$/

// 金钱格式："10000.00"、"10,000.00"、"10000"、"10,000"
/^([0-9]+|[0-9]{1,3}(,[0-9]{3})*)(.[0-9]{1,2})?$/
```
